# ThemePlate Settings

## Usage

```php
use ThemePlate\Settings;

$args = array(
	'id'     => 'test-settings',
	'title'  => 'Tester',
	'page'   => 'test-page',
	'fields' => array(),
	//'show_on'  => array(),
	//'hide_on'  => array(),
	//'context'  => 'normal',
	//'priority' => 'default',
);

new Settings( $args );
```
